This project is just to easily manipulate the vtk library


Features
--------

* Support to use objects such as:
    * vtkDoubleArray
    * vtkFloatArray
    * vtkIntArray
    * vtkIdTypeArray
    * vtkLongArray
    * vtkShortArray
    * vtkUnsignedCharArray
    * vtkUnsignedIntArray
    * vtkUnsignedLongArray
    * vtkUnsignedLongLongArray
    * vtkUnsignedShortArray
    * vtkPoints
